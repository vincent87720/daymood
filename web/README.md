# Daymood Web

## Environment
- vue@2.6.14
- vuetify@2.6.13
- vue-axios@3.4.1
- vuex@3.6.2

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
### Compiles and hot-reloads for development using docker
```
make dockerrun
```

## Build
### Compiles and minifies for production
```
npm run build
```
